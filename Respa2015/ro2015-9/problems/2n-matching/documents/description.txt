Есть 2*N участников. У каждого есть рейтинг. Чем больше рейтинг тем сильнее участник. В каждой команде два участника. Рейтинг команды это средний рейтинг двух участников. Из всевозможных разбиений, какое самое высокое и самое низкое место может получить участник под номером 1. 

1) N <= 3
2) N <= 6
3) N <= 2500
4) N <= 10^5
